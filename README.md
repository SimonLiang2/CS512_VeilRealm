# CS512-VEILREALM
CS512 SOFTWARE ENGINEERING PROJECT

### Instal Unity Hub
1. Download unity from https://unity.com/download
2. Run the installer
3. Launch Unity Hub and sign in with your account
4. If you don't have an account create one https://login.unity.com/en/sign-up

### Install Unity Engine
1. In Unity Hub
2. Go to Installs then select Install Editor
3. Install Unity 6 LTS

### Clone Repository
1. Have git installed, if not already installed follow instructions https://git-scm.com/book/en/v2/Getting-Started-Installing-Git
2. Open a terminal window and run <br> git clone https://github.com/Johnny4251/CS512_VeilRealm.git

### Open Project
1. Open Unity Hub
2. Go to Projects
3. Select Add
4. Select Add project from disk
5. Navigate to the folder/diretory that was cloned earlier
6. Open project

### Build Project
1. In Unity Engine select file
2. Select Build And Run
